# Static Website CI/CD Pipeline

## Project Overview
This project demonstrates a complete CI/CD pipeline using GitHub Actions for a static website.

## Features
- ✅ Automated deployment on push
- ✅ Version control with Git
- ✅ GitHub Actions workflow
- ✅ Deployment logs and monitoring
- ✅ Rollback capability

## Tech Stack
- Git & GitHub
- GitHub Actions
- AWS S3 / Netlify
- HTML, CSS, JavaScript
